Detailed Description

Extends QWebMethod with some generic constructors and synchronous message sending.

Subclass of QWebMethod, contains some generic methods for sending messages and various additional convenience constructors. It can save some lines of code, when compared with QWebMethod, at the expense of producing less readable code.

Can be used both synchronously (through static invokeMethod() method), or asynchronously (indicates, when reply is ready by emitting a replyReady() signal).
