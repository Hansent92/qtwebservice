Detailed Description

Used to prepare QWebMethod code for template processing.

Provides some basic methods used in preparing QWebMethod code, as well as process template tags, etc.

Generation is based on templates located in ./qtwsdlconverter/templates (for subclassed methods) and on real QWebMethod code (for non-subclassed methods).
