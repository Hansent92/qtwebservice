Detailed Description

Creates code for communicating with web services.

Generates code, based on WSDL information and user defined flags. This class does not generate web methods' code itself - it is delegated to MethodGenerator class.

Note: Generator also creates a dummy main.cpp file. It's needed only for successful compilation of freshly generated code. It is NOT NEEDED for any other reason. You can safely delete it in your project.
