    Detailed Description

    Used to prepare QWebMethod code for template processing.

    Provides some basic methods used in preparing QWebMethod code, as well as process template tags, etc.

    Generation is based on templates located in ./qtwsdlconverter/templates (for subclassed methods) and on real QWebMethod code (for non-subclassed methods).

    int TemplateLogic::removeTag ( QString & templateCode, QString tag, int beginIndex = 0 )

    Removes a given tag and returns it's begin index, which can be used to insert new data.

    Takes in templateCode string, which is the text of the file, tag, which is the tag to look for, and optionally beginIndex, which can be used to optimise the search a bit (by specifying where to start).

    QString TemplateLogic::stripFile ( QString fileData, FileType type = Header )

    Strips file data from license headers, includes etc. Takes in fileData in for of a QString, and type (it tells the algorithm what to do with the data).
