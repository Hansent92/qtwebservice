    Detailed Description

    Class providing Web Service functionality.

    This class serves as central point for use cases where multiple web methods are needed, that share certain similarities. You can store all relevant information about a web service in a single class.

    You can supply a WSDl file, which will be parsed (using QWsdl), and web methods will be extracted. You can also specify your own web methods by using addMethod().

    You can use QWebService to gather data about web service (getName(), getHost(), getMethodNames() etc.), or use it's web methods to interact with remote web service (methods(), method()).

    Especially useful is the method(), which returns a pointer to a single web method, which can be used directly to interact with QWebMethod object.

    Member Function Documentation

    QWebService::QWebService ( QObject * parent = 0 )

    Constructs the object using parent (defaults to 0). You can set a WSDL file later using setWsdl(), or add custom web methods using addMethod().

    See also setWsdl() and addMethod().

    QWebService::QWebService ( QWsdl * _wsdl, QObject * parent = 0 )

    Constructs and initializes the object with _wsdl and using parent (defaults to 0). WSDL file can be resetted later using resetWsdl(). You can add custom web methods using addMethod().

    See also resetWsdl() and addMethod().

    QWebService::QWebService ( const QString & _hostname, QObject * parent = 0 )

    Constructs and initializes the object using parent (defaults to 0). _hostname is used to initialise QWsdl if an URL or path to a valid WSDL file was used. Otherwise, WSDL can be set later with setWsdl(), or add custom web methods using addMethod().

    See also setWsdl() and addMethod().

    QWebService::~QWebService ()

    Deletes wsdl pointer.

    void QWebService::addMethod ( QWebMethod * newMethod )

    Adds the specified web method (newMethod) to the web service. Method name should be set in the object, it is used internally and for referencing.

    See also removeMethod().

    void QWebService::addMethod ( const QString & methodName, QWebMethod * newMethod )

    This function overloads addMethod().

    Adds the specified web method (newMethod) to the web service. Method name (methodName) is used for reference (it does not set name inside the QWebMethod object).

    See also removeMethod().

    void QWebService::errorEncountered ( const QString & errMessage ) [signal]

    Singal emitted when WsdlConverter encounters an error. Carries errMessage for convenience.

    QString QWebService::errorInfo () const

    Returns QString with error message in case an error occured. Otherwise, returns empty string.

    See also isErrorState().

    bool QWebService::isErrorState ()

    Returns true if object is in error state.

    QWebMethod * QWebService::method ( const QString & methodName )

    Returns a pointer to a single web method object, specified by methodName. If no method with that name exists, a default-constructed value is returned (see QMap::value() documentation for details).

    See also methods().

    QStringList QWebService::methodParameters ( const QString & methodName ) const

    Returns list of names of parameters of a method specified by methodName.

    QStringList QWebService::methodReturnValue ( const QString & methodName ) const

    Returns names of return values, for a given method (methodName).

    QMap<QString, QWebMethod *> * QWebService::methods ()

    Returns pointers to all QWebMethod objects held in QWebService, useful for invoking web methods.

    See also method().

    QMap<QString, QVariant> QWebService::parameterNamesTypes ( const QString & methodName ) const

    For a given methodName, returns a QMap with QString paramater name as a key, and QVariant value. By running QVariant::typeName() you can determine the type of the parameter.

    void QWebService::removeMethod ( const QString & methodName )

    Removes the method specified by methodName. It also deletes the underlying object, so be careful!

    See also addMethod().

    void QWebService::resetWsdl ( QWsdl * newWsdl = 0 )

    Sets the WSDL (newWsdl) file to use. This does override already present methods. If you just want to add WSDL methods to existing ones, use setWsdl()

    See also setWsdl().

    QMap<QString, QVariant> QWebService::returnValueNameType ( const QString & methodName ) const

    For a given methodName, returns a QMap with QString paramater name as a key, and QVariant value. By running QVariant::typeName() you can determine the type of the parameter.

    void QWebService::setWsdl ( QWsdl * newWsdl )

    Sets the WSDL (newWsdl) file to use. This does not override already present methods. If you want to override them, use resetWsdl().

    See also resetWsdl().
