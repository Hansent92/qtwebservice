    Detailed Description

    Class for interaction with local and remote WSDL files. Currently read-only.

    Reads web service data (message names, parameters, return values, ws name etc) from a WSDL file or URL.

    Member Function Documentation

    QWsdl::QWsdl ( QObject * parent = 0 )

    Simple constructor, requires parent only, but needs other information to be specified later in order to run. You need to run setWsdlFile(), which automatically parses the file.

    Initialises the whole object with default values.

    See also setWsdlFile() and resetWsdl().

    QWsdl::QWsdl ( QString wsdlFile, QObject * parent = 0 )

    Constructs the object using optional parent. Uses the file path or URL specified in wsdlFile to parse the WSDL file.

    QWsdl::~QWsdl ()

    Destructor - cleans some internal variables.

    void QWsdl::fileReplyFinished ( QNetworkReply * rply ) [slot]

    Asynchronous public return slot. Reads WSDL reply (rply) from server (used in case URL was specified in wsdl file path).

    Not sure why it's public (sierdzio). Maybe it will go private in the future.

    QString QWsdl::getErrorInfo ()

    Returns QString with error message in case an error occured. Otherwise, returns empty string.

    See also isErrorState().

    QUrl QWsdl::getHostUrl ()

    Quite similar tp getHostName(). The very existence of this method will be questioned :)

    See also getHostname().

    QString QWsdl::getHostname ()

    Not enirely well-thought method and variable. It returns web service's URL (probably). It will almmost certainly be rewritten or deleted in the future.

    See also getHostUrl().

    QStringList QWsdl::getMethodNames ()

    Returns a QStringList of names of web service's methods.

    See also getMethods().

    QMap<QString, QSoapMessage *> * QWsdl::getMethods ()

    Returns a QMap<QString, QSoapMessage *> pointer. Keys are method names (just as in getMethodNames()), and values are QSoapMessages themselves (which means they can be used not only to get information, but also to send messages, set them up etc.).

    See also getMethodNames().

    QString QWsdl::getTargetNamespace ()

    Returns target namespace specified in WSDL.

    QString QWsdl::getWebServiceName ()

    Returns QString with the name of the web service specified in WSDL.

    bool QWsdl::isErrorState ()

    Returns true if there was an error, false otherwise. Details about an error can be read with getErrorInfo().

    See also getErrorInfo().

    void QWsdl::resetWsdl ( QString newWsdlPath )

    Can be used to set or reset a WSDL file (or URL), using newWsdlPath. Cleans and reinitialises the object, parses the file.

    See also setWsdlFile().

    void QWsdl::setWsdlFile ( QString wsdlFile )

    Wrapper for resetWsdl(). Used to set the WSDL file or URL using wsdlFile. Compulsory after simple constructor, but not needed if you have already specified the file in the constructor or resetWsdl().

    See also resetWsdl().
