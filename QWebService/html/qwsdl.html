    Detailed Description

    Class for interaction with local and remote WSDL files. Currently read-only.

    Reads web service data (message names, parameters, return values, ws name etc) from a WSDL file. The file can be on a local filesystem, or a remote one (specified by URL).

    To get started, you have to:

    • construct QWsdl object
    • specify WSDL file or URL (can be done in constructor, or using setWsdl(), resetWsdl(),
    • check for errors using isErrorState(),
    • web methods are ready to read using methods(). Other info available from numerous getter methods (webServiceName(), targetNamespace() etc.)

    Member Function Documentation

    QWsdl::QWsdl ( QObject * parent = 0 )

    Simple constructor, requires parent only, but needs other information to be specified later in order to run. You need to run setWsdlFile(), which automatically parses the file.

    Initialises the whole object with default values.

    See also setWsdlFile() and resetWsdl().

    QWsdl::QWsdl ( const QString & wsdlFile, QObject * parent = 0 )

    Constructs the object using optional parent. Uses the file path or URL specified in wsdlFile to parse the WSDL file.

    QWsdl::~QWsdl ()

    Destructor - cleans some internal variables.

    void QWsdl::errorEncountered ( const QString & errMessage ) [signal]

    Singal emitted when WsdlConverter encounters an error. Carries errMessage for convenience.

    QString QWsdl::errorInfo () const

    Returns QString with error message in case an error occured. Otherwise, returns empty string.

    See also isErrorState().

    void QWsdl::fileReplyFinished ( QNetworkReply * rply ) [protected slot]

    Asynchronous public return slot. Reads WSDL reply (rply) from server (used in case URL was specified in wsdl file path).

    bool QWsdl::isErrorState () const

    Returns true if there was an error, false otherwise. Details about an error can be read with getErrorInfo().

    See also errorInfo().

    QStringList QWsdl::methodNames () const

    Returns a QStringList of names of web service's methods.

    See also methods().

    QMap<QString, QWebMethod *> * QWsdl::methods ()

    Returns a QMap<QString, QWebServiceMethod *> pointer. Keys are method names (just as in getMethodNames()), and values are QWebServiceMethods themselves (which means they can be used not only to get information, but also to send messages, set them up etc.).

    See also methodNames().

    void QWsdl::resetWsdl ( const QString & newWsdl )

    Can be used to set or reset a WSDL file (or URL), using newWsdl. Cleans and reinitialises the object, parses the file.

    See also setWsdlFile().
