    Detailed Description

    Creates code for web methods.

    Creates methods (both headers and sources) by generating the entire code or subclassing QWebServiceMethod (depending on whether --subclass flag is set).

    Generation is based on templates located in ./qtwsdlconverter/templates (for subclassed methods) and on real QWebMethod code (for non-subclassed methods).
