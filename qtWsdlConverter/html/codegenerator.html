    Detailed Description

    Creates code in the standard path (standard structure).

    This class will most probably be renamed, as it turns out to do almost everything in the conversion process.

    bool CodeGenerator::create ( QWsdl * wsdl, QDir workingDir, Flags * flgs, QString baseClassName = 0, QObject * parent = 0 ) [static]

    Performs the conversion in CodeGenerator. Data from WSDL (wsdl) is combined with options specified in flags (flgs), and base class name (baseClassName) to create a complete set of classes in given directory (workingDir). For Qt reasons, parent is also needed, although it defaults to 0.

    Returns true if successful.

    Singal emitted when CodeGenerator encounters an error. Carries errMessage for convenience.
